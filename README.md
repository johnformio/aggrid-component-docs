# ag-grid-mdx

A React app scaffolded with Vite and MDX support.

## Getting Started

Install dependencies:

```sh
pnpm install
```

Start the development server:

```sh
pnpm dev
```

## MDX Usage

You can import `.mdx` files directly into your React components.

## Build

```sh
pnpm build
```

## Preview

```sh
pnpm preview
```
