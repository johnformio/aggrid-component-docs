## Pagination
Pagination can be configured with in the component.  We use the ag-grid (client-side pagination model)[https://www.ag-grid.com/javascript-data-grid/row-models/#client-side] for pagintation.

<details>
    <summary>Grid configuration JSON</summary>
```json
{
  "label": "AG Grid",
  "key": "aggrid",
  "type": "aggrid",
  "agGridConfig": {
    "rowData": [...],
    "paginationPageSize": 2,
    "pagination": true
  },
  "tableView": true,
  "validateWhenHidden": false
}
```
</details>
