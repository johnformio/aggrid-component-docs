## Simple Grid
This is a simple grid with no configurations.  It Reads data from a data source.

By default the width of the grid will be 100% of the parent container.  You can set the width and height of the grid by setting parameters in the `agGridConfig` object, or constraing it with a wrappper element.

AG-Grid will automatically vertially and horizontally scroll if the grid is larger than the parent container or the specified width and height.

Sorting and filtering is enabled by default and uses the default ag-grid sorting and filtering behaviors..
